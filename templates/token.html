Option 2: Use TOTP (no daily expiry)

Set GROWW_API_KEY and GROWW_TOTP_SECRET in your environment. The app will generate a token at startup; no need to refresh daily.

Option 3: Set token manually

Generate an access token from your Groww Trading APIs profile and set GROWW_ACCESS_TOKEN in your environment. Token expires daily at 6:00 AM.
